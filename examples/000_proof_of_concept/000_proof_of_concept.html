Waveshaping

Waveshaping is a signal processing technique that maps an input signal to an output signal through the use of a transfer function. The transfer function's domain must be well defined on the range of the output signal. The technique is useful for distortion and soft clipping, as the limit of the transfer function could approach -1 and 1 as the input signal approaches -infinity and infinity respectively.

Mathematically, waveshaping is a simple operation. It is defined as follows: w(t) = f(i(t)x(t)) where x(t) is the input signal, f(t) is the transfer function and i(t) is an index function (which acts more or less as a numerical gain). If i(t)x(t) has the range [a, b] then f(t) must be well defined on this domain.
